FateUtils - Discord Bot

O FateUtils é um bot criado para auxiliar em jogos Fate via Discord, rolando dados ou gerando estatísticas aleatórias diversas.
Para listar todos os comandos do bot utilize o comando !help

Adicionar ao seu servidor

Para adicionar o FateUtils ao seu servidor Discord, clique no link abaixo ou copie-o e cole no seu navegador:
https://discordapp.com/api/oauth2/authorize?client_id=695613875168608336&permissions=0&scope=bot

Como usar

Em todos os comandos os parâmetros são opcionais, utilize-os sem os parênteses.

Rolagens de dados

!r (o seu bônus) (o número alvo) (a ação desejada)

  • !r = rola 4 dados fate
  • !r +3 = rola 4 dados fate + bônus
  • !r +3 5 = rola 4 dados fate + bônus e o número alvo, indicando o resultado
  • !r +3 5 A = faz a rolagem para a ação especificada e já dá o resultado completo
  • As ações: (A)taque | (D)efesa | (S)uperar | (CA) Criar vantagem para Criar Aspecto | (CE) Criar vantagem em Aspecto Existente

Ao invés de !r você pode especificar uma ação de Fate:

  • !atacar (o seu bônus) (o valor da defesa) = para atacar
  • !defender (o seu bônus) (o valor do ataque) = para se defender
  • !superar (o seu bônus) (a dificuldade) = para superar
  • !criar (o seu bônus) (a dificuldade) = para criar um aspecto
  • !descobrir (o seu bônus) (a dificuldade) = para descobrir um aspecto existente

!edge = rola 3 dados fates +1d6, aceita os mesmo parâmetros de !r

!tachyon (tipo) = rola 6 dados fates e seleciona 4, aceita os mesmo parâmetros subsequentes de !r.
O parâmetro tipo indica se serão escolhidos os 4 melhores (V) ou 4 piores (D). (ex.: !tachyon V +3)

Estatísticas aleatórias

!pdn (nome) = gera um personagem da narradora aleatório, o primeiro parâmetro será o seu nome (ex.: !pdn José)

!abordagens = gera valores aleatórios para as abordagens FAE

!pericias (topo) = gera valores aleatórios para as perícas FCon, o primeiro parâmetro é o topo da pirâmide (ex.: !pericias 4)

!aspecto (tipo) = aceita os tipos conceito e problema, se não especificado retorna um aspecto genérico

!facanha (tipo) = como tipo aceita qualquer perícia ou abordagem

Fichas completas

!criatura = gera uma criatura aleatória, baseada no livro Fate Criaturas Fantásticas, da Pluma Press

!uranium = gera um personagem aleatório completo de Uranium Chef

!umdaar = gera um personagem aleatório completo de Mestres de Umdaar

!umdaar aventura = gera uma aventura aleatória para Mestres de Umdaar

!medieval = gera um personagem aleatório completo de fantasia medieval
!medieval (sistema) (bioforma) (classe) = ex.: !medieval fcon halfling ladino

Os parâmetros possíveis são (para bioformas ou classes aleatórias coloque um x):
sistema = fcon ou fae
bioforma = elfo, anao, halfling ou humano
classe = druida, ladino ou monge

Utilidades

!fp = mostra a quantidade de Pontos de Destino (*fate points*) que você possui

!fp (valor) = define a quantidade de PDs que você possui, aceita qualquer valor entre 0 e 99

!fp+ = aumenta a sua quantidade de PDs em um

!fp- = diminui a sua quantidade de PDs em um

!utils store = grava no bot o resultado do seu último comando

!utils show = mostra o conteúdo que está gravado
